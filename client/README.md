# OpenModels Client

This directory contains the React frontend codebase for **OpenModels**.

For full information on getting started, configuring environment variables, running the local development environment, and production deployment, please refer to the **[Main Root README.md](../README.md)**.

## 🛠️ Client Tech Stack
- **Framework**: React 19 + TypeScript
- **Build Tool**: Vite 8
- **Styling**: Tailwind CSS v4 (using the `@tailwindcss/vite` compiler plugin)
- **State Management**: Zustand
- **Code & Preview**: Monaco Editor + Sandboxed Preview Iframe
